# Qsmtool User Guide

## Install & Setup
### Install
    1. 解壓安裝文件包,將Qsmtool放置到安裝目錄
    2. 設置環境變量
        - QSMT_HOME    : 工具所在目錄
        - CFG_SAVE_DIR : 輸出的配置文件保存目錄
        - MAIN_CFG_FILE: 輸入的主配置文件
        - USR_CFG_FILE : 輸入的用戶配置文件
        - CBS_HOME     : 驗證環境主目錄（可根據需求隨意設置，作爲文件瀏覽的根目錄）
    It's OK!

### Setup
    run cmd 'qsmtool' in terminal.
    >$ qsmtool
